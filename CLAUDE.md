# WorkinClaude Project Conventions

## Overview

This is the WorkinClaude plugin repository - a team collaboration skill for Claude Code.

## Tech Stack

- **Skill Framework**: Claude Code skills
- **Templates**: Markdown-based
- **Scripts**: Node.js

## Workflow

This project uses the following workflow:
- Brainstorming + TDD for planning and development
- Subagent-driven development for implementation
- Team memory for context preservation

Design specs: `docs/superpowers/specs/`

Default skills bundled:
- `karpathy-guidelines` - Coding guidelines
- `systematic-debugging` - Debug methodology
- `writing-skills` - Skill authoring
